# Mongo auth by url

./mongou mongodb://user:pass@host:port/db must be run mongo -u user -p pass host:port/db


# Install
```
git clone https://github.com/dutchakdev/mongourl.git
cd ./mongourl
cp mongou /usr/local/bin/
chmod +x /usr/local/bin/mongou
```
